QUICK NOTES — Cómo instalar la extensión
==========================================

1. Descarga o clona el repositorio de GitHub

2. Abre tu navegador:
   - Chrome: escribe en la barra  chrome://extensions
   - Opera:  escribe en la barra  opera://extensions

3. Activa el "Modo desarrollador"
   (interruptor arriba a la derecha)

4. Haz clic en:
   - Chrome: "Cargar descomprimida"
   - Opera:  "Cargar extensión sin empaquetar"

5. Selecciona la carpeta "quick-notes"
   (la que contiene manifest.json, popup.html, popup.css, popup.js)

6. ¡Listo! Aparece el ícono en la barra del navegador.
   Haz clic en él para abrir Quick Notes.

NOTA: No cierres ni muevas la carpeta o la extensión dejará de funcionar.
